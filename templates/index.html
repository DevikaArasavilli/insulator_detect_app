Welcome to InsightGuard

InsightGuard is an AI-powered web application designed to detect faults in high-voltage power line insulators. By analyzing images from multiple perspectives, it helps ensure:

  • ⚡ Preventive maintenance of power infrastructure
  • 📉 Reduction in manual inspection costs and errors
  • 🔍 Early detection of physical or structural damage

Upload images from all four views of an insulator – front, back, left, and right – and our intelligent model will evaluate each side for defects.

How It Works

  1. Upload Images: Provide four images covering all angles – Front, Back, Left, and Right.
  2. AI Model Processing: Each image is analyzed by a trained Convolutional Neural Network (CNN).
  3. Per-View Diagnosis: Visual results for each view are shown with confidence scores and defect highlights.
  4. Final Verdict: If any one view is defective, the overall insulator is marked as “Defective”.

Results are delivered within seconds after upload – making it fast and reliable for engineers and field operators.

About InsightGuard

This project was developed by a team of passionate engineers aiming to modernize and automate power system inspections using deep learning and computer vision.

Highlights of the project:

  • 🔬 Trained on a curated dataset of insulator images
  • 🤖 CNN model built with TensorFlow/Keras
  • 🖼️ Image preprocessing with OpenCV for accuracy
  • 🌐 Deployed using Flask backend and interactive HTML/CSS frontend

The system is scalable and adaptable for real-time deployment with drone image feeds and SCADA integrations in future versions.
